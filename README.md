# BloodLink 🩸

**BloodLink** is a smart, real-time blood donation coordination platform built specifically for Karachi, Pakistan. It connects donors, hospitals, and coordinators seamlessly to ensure critical blood needs are met quickly and efficiently. 

The application features a modern interface to manage active blood requests, register donors, and coordinate logistics. It also includes an integrated **AI Assistant** (powered by Anthropic's Claude) capable of understanding and responding in Urdu, Roman Urdu, and English. The AI agent provides instant answers regarding donor availability, urgent hospital needs, and registration guidance.

### ✨ Features
- 🤖 **AI Chat Agent**: Multilingual AI assistant to help users with queries in real-time.
- 🏥 **Hospital Connect**: Track and manage critical blood requests from different hospitals.
- 🧑‍🤝‍🧑 **Donor Registration**: Easy onboarding for new donors with location-based grouping.
- 📊 **Coordinator Dashboard**: Real-time stats and management tools for platform administrators.
